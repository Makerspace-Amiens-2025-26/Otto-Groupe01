# Conception et prototypage
Conception des pièces

Nous avons réalisé des pièces personnalisées sur Onshape à partir du modèle d'origine Otto-MKS. Les éléments que nous avons modifiés sont :

Un casque et des pieds reprenant l’apparence d’un chat
Des bras articulés connectés entre eux
Impression 3D

Après avoir finalisé la modélisation, nous avons fabriqué les différentes pièces à l’aide des imprimantes 3D du MakerSpace.

Programmation

Le robot a ensuite été programmé avec Arduino. Le programme permet notamment :

De faire se déplacer le robot (avancer, tourner, etc.)
De le piloter en Bluetooth grâce à l’application RemoteXY
D’utiliser un mode autonome lui permettant d’avancer sans télécommande
De commander les bras articulés pour effectuer des attaques
